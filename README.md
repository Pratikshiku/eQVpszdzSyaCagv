## Second-hand Item Trading

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/22f791dcf3f242eab6daf858a89dd4f4/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Merchant, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Announcements
5. Merchants
6. Products
7. Personal Center
8. Delivery Address
9. Product Favorites
10. Product Orders


**Merchant Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Product Management
6. Product Review Management
7. Product Favorites Management
8. Product Order Management
9. User Inquiry Management
10. Announcement Management
11. Carousel Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Merchant Management
5. User Management
6. Product Management
7. Product Review Management
8. Product Review Management
9. Product Order Management
10. User Inquiry Management
11. Announcement Management
12. Basic Data Management (Announcement Type Management, Merchant Credit Type Management, Product Type Management, Data Type Management, Status Management)
13. Carousel Management
